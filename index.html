SceneHub

Scalable JSON-Based WebGL Asset Repository

Build 3D scenes fast from a library of reusable assets



Features

  • Scalable

    An extensible library of JSON-encoded WebGL assets

  • Build a scene anywhere

    Hotlink the API script into your page or blog and start building scenes containing assets. The assets are loaded on-demand and cross-domain - just reference them by ID.

  • SceneJS on rails
    The API is a wrapper for SceneJS, giving you a powerful SceneJS-on-rails experience. You can skip the asset library altogether and just use the API as a stand-alone WebGL engine.

  • Plug and play

    Asset reuse at multiple levels: attributes, objects, object graphs, and even entire scenes

  • Open Source
    Licensed under both the GPL and MIT licenses - pick whichever.

  • Fork it

    Fork the repository and add your own assets. Want to share them? Submit a pull request.

Quick Start

First, hotlink to the SceneHub API library: 

<script src="http://xeolabs.github.com/scenehub/api/v0/scenehub.js"></script>

Then build a scene - we're going to build an articulated object hierarchy using one geometry asset and a couple of surface appearance assets:


var scenehub = new SceneHub();

var scene = scenehub.createScene({
    camera:{
        eye:{ z:-100 }
    },
    objects:[
        {
            translate:{ y:5 },
            rotate: { y : 1, angle: 45 },
            objects:[
                {
                    id: "superman",
                    useGeometry:"prims.box",
                    useAppearance:"heroes.superman",
                    scale:{ x:10, y:10, z:10 },
                    translate:{ x:-10 }
                },
                {
                    id: "generalZod",
                    useGeometry:"prims.box",
                    useAppearance:"villains.generalZod",
                    scale:{ x:10, y:10, z:10 },
                    rotate:{ x:1, y:1, z:1, angle:-20 },
                    translate:{ x:10 }
                }
            ]
        }
    ]
});
