Build WebGL graphics fast

An open source JSON-based engine with sharable assets

Features

  • Build a scene in minutes

    An SDK built on SceneJS that uses convention-over-configuration for easy scene definition.

  • Asset repository

    A default backend repository of generic assets, loaded on-demand as you reference them in your scenes. The repository is hosted at Firebase and mirrored at GitHub, where people can contribute assets for peer review.

  • DRY to the max

    Almost anything can be an asset: appearances, geometries, lights, cameras, objects, and even entire scenes.

  • Repository is optional

    You're free to define your own custom assets with the SDK, while falling back on the repository for generic things like skyboxes. You can also configure the SDK to load assets from your own JSON source, or even just not use a repository altogether.

  • Open Source
    Licensed under both the GPL and MIT licenses - pick whichever you need.

Quick Start

First, include our JavaScript library in the <head> tag of your web app: 

<script src="http://scenehub.github.io/scenehub/api/v0/scenehub.js"></script>

Then build a scene. We're going to build an articulated object hierarchy using three types of asset: a scene object, a geometry and a surface appearance: 


var scenehub = new SceneHub();

var scene = scenehub.createScene({

    camera:{
        eye:{ z:-100 }
    },

    objects:[
        // Instance a library object (a skybox) and override its scale
        {
            useObject:"skies.clouds",
            scale: { x: 1000, y: 1000, z: 1000 }
        },
        // Group object
        {
            translate:{ y:-10 },
            objects:[
                 // Object using geometry and appearance assets
                {
                    id: "superman",
                    useGeometry:"prims.box",
                    useAppearance:"heroes.superman",
                    scale:{ x:10,y:10, z:10 },
                    rotate: { y: 1, z: 1, angle: 30 }
                }
            ]
        }
    ]
});

Now let's start the cube spinning:

// on() is used to listen for data changes on scene elements.
// This is the primary way to read data from SceneHub.
scene.on("objects.superman", function (superman) {

      var angle = 0;

      scene.on("tick", function() {
        superman.set({
          rotate: { angle: angle += 0.5 }
        });
      });
    });
