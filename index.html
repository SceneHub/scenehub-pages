Scalable JSON-Based WebGL Asset Repository

Build 3D scenes fast from a reusable library of geometries, shaders, textures and sounds



First, link to the SceneHub API library in your page:

<script src="http://xeolabs.github.com/scenehub/api/v0/scenehub.js"></script>

Then get yourself a client and build a scene. We're going to build an articulated object hierarchy using one geometry asset and a couple of surface appearance assets:


var scenehub = new SceneHub();

var scene = scenehub.createScene({
    camera:{
        eye:{ z:-100 }
    },
    objects:[
        {
            translate:{ y:5 },
            rotate: { y : 1, angle: 45 },
            objects:[
                {
                    id: "superman",
                    useGeometry:"prims.box",
                    useAppearance:"heroes.superman",
                    scale:{ x:10, y:10, z:10 },
                    translate:{ x:-10 }
                },
                {
                    id: "generalZod",
                    useGeometry:"prims.box",
                    useAppearance:"villains.generalZod",
                    scale:{ x:10, y:10, z:10 },
                    rotate:{ x:1, y:1, z:1, angle:-20 },
                    translate:{ x:10 }
                }
            ]
        }
    ]
});
