Build WebGL scenes fast

A JSON-based 3D engine with a shared public asset repository

Features

  • Build a scene in minutes

    Just load the API library and start building 3D scenes in JSON. The API is a richly-featured WebGL engine built around SceneJS V3 that follows convention-over-configuration to provide a slick "SceneJS on rails" experience.

  • Plug and play

    By default, the API is hooked into a GitHub repository of generic JSON-encoded WebGL assets. When you reference those assets by their IDs, the API will transparently fetch them cross-domain. Sweet for code sharing.

  • DRY to the max

    Almost anything can be an asset: appearances, geometries, lights, cameras, objects, and even entire scenes.

  • Repository is optional

    You're free to define your own custom assets with the API, while falling back on the repository for generic things like skyboxes. You can also configure the API to load assets from your own JSON source, or even just not have a repo.

  • Open Source
    Licensed under both the GPL and MIT licenses - pick whichever you need.

  • Fork it

    Fork the repository and add your own assets. Want to share them? Submit a pull request.

Quick Start

First, hotlink to the SceneHub API library: 

<script src="http://xeolabs.github.com/scenehub/api/v0/scenehub.js"></script>

Then build a scene. We're going to build an articulated object hierarchy using three types of asset: a scene object, a geometry and a surface appearance: 


var scenehub = new SceneHub();

var scene = scenehub.createScene({

    camera:{
        eye:{ z:-100 }
    },

    objects:[
        // Instance a library object (a skybox) and override its scale
        {
            useObject:"skies.clouds",
            scale: { x: 1000, y: 1000, z: 1000 }
        },
        // Group object
        {
            translate:{ y:-10 },
            objects:[
                 // Object using geometry and appearance assets
                {
                    id: "superman",
                    useGeometry:"prims.box",
                    useAppearance:"heroes.superman",
                    scale:{ x:10,y:10, z:10 },
                    rotate: { y: 1, z: 1, angle: 30 }
                }
            ]
        }
    ]
});

Now let's start the cube spinning:

// on() is used to listen for data changes on scene elements.
// This is the primary way to read data from SceneHub.
scene.on("objects.superman", function (superman) {

      var angle = 0;

      scene.on("tick", function() {
        superman.set({
          rotate: { angle: angle += 0.5 }
        });
      });
    });
