Scalable JSON-Based WebGL Asset Repository

Build 3D scenes fast on a library of reusable assets



Features

  • Scalable

    An extensible library of JSON-encoded WebGL assets

  • Build a scene in minutes

    Hotlink to the API script and build scenes in JSON. As you reference assets the API loads them on-demand, cross-domain. Great for code sharing.

  • SceneJS on rails
    The API is built around SceneJS V3 and follows convention-over-configuration to provide a very slick scene building experience. Skip the asset repository altogether if you want to manage the assets yourself and use the API as a stand-alone WebGL engine.

  • Plug and play

    Almost everything is a reusable asset: attributes, objects, object graphs, and even entire scenes.

  • Open Source
    Licensed under both the GPL and MIT licenses - pick whichever you need.

  • Fork it

    Fork the repository and add your own assets. Want to share them? Submit a pull request.

Quick Start

First, hotlink to the SceneHub API library: 

<script src="http://xeolabs.github.com/scenehub/api/v0/scenehub.js"></script>

Then build a scene. We're going to build an articulated object hierarchy using three types of asset: a scene object, a geometry and a surface appearance: 


var scenehub = new SceneHub();

var scene = scenehub.createScene({

    camera:{
        eye:{ z:-100 }
    },

    objects:[
        {
            // Reuse a library object (a skybox), override its scale
            useObject:"skies.clouds",
            scale: { x: 1000, y: 1000, z: 1000 }
        },
        {
            translate:{ y:-10 },
            objects:[
                // Reuse geometry and appearance assets
                {
                    id: "superman",
                    useGeometry:"prims.box",
                    useAppearance:"heroes.superman",
                    scale:{ x:10,y:10, z:10 },
                    rotate: { y: 1, z: 1, angle: 30 }
                }
            ]
        }
    ]
});

Note that you can also create assets through the API, so you're not actually dependent on the asset library.Now let's spin the cube around:

//
scene.on("objects.superman", function (superman) {

      var angle = 0;

      scene.on("tick", function() {
        superman.set({
          rotate: { angle: angle += 0.5 }
        });
      });
    });
